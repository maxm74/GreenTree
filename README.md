GreenTree is an utility for running scheduled tasks and more that is placed in the tray area.
Currently the shutdown operations only work on Windows, but a porting to other platforms will be done soon.

# Prerequisites to Compile

GreenTree use the following packages, you need to install them in Lazarus before you can compile it

* MaxM Commons at https://github.com/maxm74/MaxM

you also need to use the latest version of Lazarus with fpc at least version 3.3.1
